## Getting Started

Instalar dependências do back-end

```
cd backend && npm install
```

Executar back-end

```
npm run dev
```
Instalar dependências do front-end web

```
cd web && npm install
```

Executar front-end web

```
npm start
```
Instalar dependências do app mobile

```
cd mobile && npm install
```

Executar mobile

```
npm start
```
